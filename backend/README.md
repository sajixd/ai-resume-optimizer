# AI Resume Optimizer - Backend

This is the FastAPI backend for the AI Resume Optimizer.

## Technologies Used
- FastAPI
- Groq (Llama 3.3-70B)
- Supabase
- PyMuPDF (fitz)

## Setup
1. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the server:
   ```bash
   fastapi dev main.py
   ```

Refer to the root [README.md](../README.md) for full project documentation.
