# AI Resume Optimizer - Frontend

This is the React frontend for the AI Resume Optimizer.

## Technologies Used
- React 19
- Vite
- Tailwind CSS 4
- Lucide React

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

Refer to the root [README.md](../README.md) for full project documentation.
